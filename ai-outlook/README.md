# AI 시대 직업 소멸 위험도 분석

한국 직업 537개의 **AI 자동화 노출 위험도를 0–100 점수로 정량화**하는 데이터 파이프라인 프로젝트입니다.

## 문제정의
국내엔 직업별 AI 노출도 데이터가 없어, 데이터가 풍부한 미국 직업 데이터를 **직업 매칭으로 이식**해 위험도를 추정합니다.

```
한국 직업 537개 ──(이름 매칭)──▶ 미국 직업(SOC) ──▶ 미국 AI지표 이식 ──▶ 위험점수 0–100
                                                                          │
                                          한국 자체 전망(증가/유지/감소)으로 검증
```

## 데이터
- **미국**: 직업 마스터(GPT-4 노출도·Frey-Osborne 자동화확률), BLS 2024–34 고용전망(occupation.xlsx), OEWS 2025년 최신 임금/고용
- **한국**: 직업전망(KECO 코드 + 전망 라벨), 537개 직업
- ⚠️ 데이터는 용량·출처 문제로 저장소에 미포함

## 분석 과정
1. **미국 데이터 정합성 점검 & 병합** — 3종 데이터의 SOC 코드를 점검하고, 2018년 SOC 분류 개정으로 생긴 코드 불일치(통합·잔여 카테고리)를 직접 확인·매핑해 단일 마스터(798×49)로 병합
2. **한국→미국 직업 매칭** — 분야를 넘는 오매칭 문제를 **① 카테고리 게이팅**(같은 직업 분야 내에서만 후보 비교) + **② 이중언어 임베딩**(한국어 원문·영어 번역 의미를 함께 활용)으로 해결 → 자동확정 215→481개
3. **위험점수 설계** — AI 노출·물리 자동화·고용전망 3채널 가중합으로 0–100 점수 산출
4. **외부 검증** — 한국 자체 전망 라벨과 대조해 방향성 확인
5. **EDA** — 위험점수 분포 / KECO 분야별 위험도 / 고위험·저위험 TOP·BOTTOM 직업 / 위험 vs 임금·학력 관계

## 결과
- 미국 데이터 분류 개정 불일치를 정리해 신뢰할 수 있는 마스터 데이터셋 구축
- 매칭 자동확정률 **약 40% → 90%** 개선
- 한국이 "감소"로 본 직업의 위험점수가 "증가"보다 확연히 높아 **방향성 검증 성공**
- ⚠️ 핵심 한계: gpt4 노출도는 "AI가 그 일을 할 수 있나"이지 "직업이 사라지나"가 아니므로, 점수는 **"AI 자동화 노출도"**로 해석하는 것이 적절

## 사용 기술
Python, pandas, 의미 임베딩 기반 매칭, rapidfuzz, matplotlib, seaborn

# AI 시대 직업 소멸 위험도 분석 (진행 중)

한국 직업 537개의 **AI 자동화 노출 위험도를 0–100 점수로 정량화**하는 데이터 파이프라인 프로젝트입니다.

## 문제정의
한국엔 직업별 AI 노출도 데이터가 없어, 데이터가 풍부한 미국 지표를 직업 매칭으로 이식해 위험도를 추정합니다.

## 데이터
- 한국 직업전망(KECO) + 미국 BLS/O*NET/연구지표 (GPT-4 노출도, Frey-Osborne 자동화확률)
- 한국 537개 직업 × 25개 지표

## 분석 과정
1. 한국 → 미국 직업(SOC) 이름 매칭
2. 오매칭 문제를 **카테고리 게이팅 + 이중언어 임베딩**으로 해결 (자동확정 215→481개, 약 40%→90%)
3. AI노출·자동화·고용전망 3채널 가중합 위험점수 설계
4. 한국 자체 전망 라벨로 외부 검증

## 결과
- 매칭 자동확정률 대폭 개선
- 한국이 "감소"로 본 직업의 위험점수가 "증가"보다 확연히 높아 **방향성 검증 성공**
- 현재 EDA·노트북 정식 반영 진행 중

## 사용 기술
Python, pandas, 임베딩 기반 매칭, rapidfuzz

# 콜센터 콜량 수요예측

콜센터 4개 포트폴리오의 인입 콜량(CV)·응대시간(CCT)·포기율(ABD)을 **30분 단위로 예측**하는 모델을 개발했습니다.

## 문제정의
콜센터는 시간대별 콜량 변동이 커 인력 배치가 어렵습니다. 정확한 수요예측으로 **적정 상담원 배치 의사결정의 근거**를 마련하는 것이 목표입니다.

## 데이터
- 콜센터 운영 데이터 (데이터 경진대회 제공)
- 일별: 2024.01–2025.12 / 30분 단위: 2025.04–06
- ⚠️ 데이터는 라이선스상 저장소에 미포함

## 분석 과정
1. 전처리 (결측치는 train 통계로만 채워 데이터 누수 방지)
2. EDA (요일·시간대·계절성·상관관계)
3. 피처 엔지니어링 (Lag·Rolling·Cyclic encoding·공휴일·staffing)
4. 포트폴리오별 LightGBM 독립 학습 (log 변환·early stopping)
5. Interval 예측을 Top-Down 분배 → End-to-End 직접예측으로 개선

## 결과
- 일별 콜량(CV) 예측 **MAPE 3.7–8.3%**, 응대시간(CCT) **2.4–3.4%**
- 시간대별 적정 인력 배치 의사결정 근거 제공

## 사용 기술
Python, pandas, LightGBM, matplotlib, seaborn
